# Requirements

- Node.js >= 18 (LTS recommended)
- npm >= 9

## Quick Start
`ash
# Clone
git clone git@github.com:Vedantpatil05/jobscraper.git
cd jobscraper

# Ensure Node version (if using nvm)
nvm install 18
nvm use 18

# Install
npm ci

# Build
npm run build

# Run
npm start
`

## Development
`ash
npm run dev
`

## Environment
- Optional: create .env to override settings. See src/config.ts for options.
